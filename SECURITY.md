# Security Policy

## ⚠️ Warning

Voice Code executes LLM-generated code and shell commands on your machine. The permission system restricts what the agent can do, but **use at your own risk**.

- Always review tool operations before confirming in default mode
- `--permission-mode bypass` bypasses all checks — only use in trusted environments
- Never run with untrusted or unverified prompts

## Reporting

If you find a security vulnerability, open an issue on GitHub. Do not post it publicly until it is addressed.
